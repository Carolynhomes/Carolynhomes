# 第二题-消灭卡片

该题主要是来解决一个整数分解的问题，判断一个数 n 是否可以表示成 5*y+3*x 的形式，并且返回 x+y 的最小值。

```python
def solution1():
    t = int(input())
    for _ in range(t):
        n = int(input())
        res = -1

        # 3x + 5y = n
        # 遍历 5的个数，最多n//5个，不然就超过n了
        for i in range(n // 5, -1, -1):
            z = n - i * 5
            if z % 3 == 0:
                res = i + z // 3
                break
        print(res)
```

`for y in range(n // 5, -1, -1)`：之所以从n//5开始，从大到小遍历，是因为，既然可以表示成5*y+3*x，那么肯定比该数除以5大了

```python
def solution2():
    t = int(input())
    
    # 初始化字典来管理招募人数和最大值
    # 当然这个地方用列表也是可以的，gpt想装一下？哈哈哈哈哈
    # count = [0] * 6(5)
    counts = {i: 0 for i in range(1, 6)}
    max_counts = {i: 0 for i in range(1, 6)}

    for _ in range(t):
        number_list = list(map(int, input().split()))
        
        for i in range(1, 6):
            if i in number_list:
                counts[i] += 1  # 如果当前数字在该行，连续计数加一
            else:
                # 更新最大值并重置连续计数
                max_counts[i] = max(max_counts[i], counts[i])
                counts[i] = 0

    # 检查最后一次连续计数
    for i in range(1, 6):
        max_counts[i] = max(max_counts[i], counts[i])

    # 打印结果
    print(" ".join(str(max_counts[i]) for i in range(1, 6)))


solution2()
```
